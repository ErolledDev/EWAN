Widget Installation Example

This page demonstrates how the chat widget appears when installed on a website. The widget should appear in the bottom right corner of the page.

How to Install on Your Website

To add this chat widget to your own website, copy and paste the following code just before the closing </body> tag:

<script src="https://widget-chat-ai.netlify.app/widget/chat.js"></script>

<script>
  new BusinessChatPlugin({
    uid: 'YOUR_USER_ID'
  });
</script>

Replace YOUR_USER_ID with your user ID from the dashboard.
